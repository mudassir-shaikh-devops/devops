Linux Process Management

1. What is a process?
A running program in Linux.

2. Commands:
- ps → show processes
- top → live process monitoring
- htop → advanced view
- kill PID → stop process
- kill -9 PID → force kill

3. Background processes:
- & → run in background
- jobs → show background jobs
- fg → bring process to foreground

4. Example:
- sleep 100 &
- kill 1234
